# Bastify---Der-ultimative-All-in-One-Desktop-Music-Player
Bastify ist ein moderner, in Python entwickelter Musikplayer, der deine lokale Bibliothek nahtlos mit der Welt des Streamings verbindet. Mit einer eleganten Benutzeroberfläche (basierend auf CustomTkinter) und leistungsstarken Features bietet Bastify alles, was du für den perfekten Musikgenuss am PC brauchst.
